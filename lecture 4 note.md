# 4주차 셸 명령어 요약 📝

---

### **1. 기본 탐색**
- `pwd` : 현재 위치 경로 확인
- `cd [경로]` : 디렉토리 이동
- `ls` : 파일 및 디렉토리 목록 보기

---

### **2. 파일/디렉토리 관리**
⚠️ **주의: `rm` 명령어는 파일을 영구적으로 삭제합니다.**
- `cp [원본] [대상]` : 파일/디렉토리 복사
- `mv [원본] [대상]` : 파일/디렉토리 이동 또는 이름 변경
- `rm [파일/디렉토리]` : 파일/디렉토리 삭제
- `mkdir [디렉토리명]` : 새 디렉토리 생성

---

### **3. 유용한 기능**
- `clear` : 화면 지우기
- `help [명령어]` : 명령어 도움말 보기
- `man [명령어]` : 명령어 매뉴얼 보기
- `exit` : 터미널 종료
- `Tab` 키 : 자동 완성
- `↑` (위쪽 화살표) : 이전 명령어 불러오기

---

### **4. 와일드카드**
- `*` : 모든 문자
- `?` : 하나의 문자

**예시:** `rm *.txt` (모든 .txt 파일 삭제)
